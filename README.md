## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/LeoScripts/beacademy-devstart-php-db

# Access
$ cd beacademy-devstart-php-db

# run project access
cd projeto-final

# rename file .env.example to .env
# insert your credentils in
# database and password

#create tables
# execute data of file db_store.sql in your datatase for create the tables and create rows in database

# run database
make up

# (optional) view SGBD Adminer 4.8.1
# acces localhost:8080

# run server
php -S localhost:8000 -t public

# access your browser localhost:8000
```
